Спринт

Мини-игра «Спринт» - это тренировка для повторения слов.

Перед вами игра "Спринт". Вы увидете карточку со словом на английском языке и перевод. Вам нужно понять, соответствует ли перевод слову на английском. Если соответстует - жмите кнопку "ВЕРНО" (или клавишу 1); если не соответствует - жмите кнопку "НЕВЕРНО" (или клавишу 2). Спешите - у вас есть только 60 секунд! Угадывайте несколько слов подряд и получайте больше очков. Трижды за игру вы можете воспользоваться подсказкой - жмите кнопку "Подсказка" (или клавишу 3).
